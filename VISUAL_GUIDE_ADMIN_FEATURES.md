## User Flow Diagrams

### Customer Edit Flow:
```
Start
  ?
Login as Admin
  ?
Navigate to Customers
  ?
Click Customer ? View Details
  ?
Click "Edit Customer" Button
  ?
Edit Dialog Opens (pre-filled)
  ?
Modify Fields (Name, Phone, Tier, etc.)
  ?
Click "Save Changes"
  ?
[Validation]
  ? valid
API Call: PUT /api/users/{id}/customer
  ? success
Show Success Message
  ?
Reload Customer Data
  ?
Dialog Closes
  ?
Updated Info Displayed
  ?
End
```

### Employee Creation Flow:
```
Start
  ?
Login as Admin
  ?
Navigate to /admin/employees
  ?
Click "Add Employee" Button
  ?
Create Dialog Opens
  ?
Fill Form:
  - Name, Email, Phone
  - Password, Role
  ?
Click "Create Employee"
  ?
[Validation]
  ? valid
API Call: POST /api/users/employees
  ? success
Show Success Message
  ?
Reload Employee List
  ?
Dialog Closes
  ?
New Employee Appears in Table
  ?
End
```
